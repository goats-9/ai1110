\begin{problem}[1. iii] 
Solve: $3\tan^{-1}x + \cot^{-1}x = \pi$.
\end{problem}

\solution 
The given inverse trigonometric equation is
\begin{equation}
\label{eq}
3\tan^{-1}x + \cot^{-1}x = \pi
\end{equation}
Also, we know the inverse trigonometric identity
\begin{equation}
\label{id}
\tan^{-1}x + \cot^{-1}x = \frac{\pi}{2}
\end{equation}
Using \eqref{id} in \eqref{eq}, we get 
\begin{align}
2\tan^{-1}x = \frac{\pi}{2},
\\
\implies \tan^{-1}x = \frac{\pi}{4},
\\
\implies x = 1.
\label{sol}
\end{align}
Therefore x = 1 is the only solution to the equation, as is clear from the graph shown below (The solution is also verified in the C source file \texttt{./codes/1{\_}3.c}).
