\noindent \textbf{Exercise 15.1, Q2.} 1500 families with 2 children were selected randomly, and the following data were recorded:

\begin{table}[!htb]
\input{tables/table.tex}
\label{tab1}
\end{table}

\noindent Compute the probability of a family, chosen at random, having
\begin{enumerate}[label=(\roman{enumi})]
\item 2 girls
\item 1 girl
\item No girl
\end{enumerate}

\noindent Also, check whether the sum of these probabilities is 1.

\solution Denote the outcome of the experiment by a random variable $X\in \cbrak{0, 1, 2}$, where $X = i$ denotes that the chosen family has $i$ girls, $i\in\cbrak{0, 1, 2}$. Then (to 3 d.p.),
\begin{align}
\pr{X = 0} = \frac{211}{1500} = 0.140 \\
\pr{X = 1} = \frac{814}{1500} = 0.543 \\
\pr{X = 2} = \frac{475}{1500} = 0.317
\label{sol}
\end{align}

\noindent One can also verify that since these events are mutually exclusive and exhaustive, we get $\pr{X = 0} + \pr{X = 1} + \pr(X = 2) = 0.140 + 0.543 + 0.317 = 1$. The Python code \texttt{./codes/5{\_}1.py} computes said probabilities after generating random samples, verifying that their sum is 1 (in the limit of float precision).
